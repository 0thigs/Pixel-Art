
# Pixel-Art App

A basic Pixel Art drawing app made with: 

- **HTML**
- **CSS**
- **JAVASCRIPT**

#### Preview:

![](https://i.imgur.com/Yg18fYa.png)

![](https://i.imgur.com/Kw3e1xI.png)

#### Functionalities:

- Choose a color from a vast palette;
- Paint the board with the color you choose;
- Clear the board.
